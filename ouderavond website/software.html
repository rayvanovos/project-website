software

we gebruiken verschillende software voor school

Visual studio code Trello Figma Microsoft Teams GitHub
Visual studio code is wat we gebruiken om te progameren. Trello gebruiken we om projecten te plannen. Figma gebruiken we om websites te ontwerpen. we gebruiken Teams om te communiceren met mede studenten en docenten. Ook gebruiken we Teams tijdens online les om uitleg te volgen. GitHub gebruiken we om samen te werken aan een project.
in visual studio code kan geprogameerd worden in verschillende talen zoals HTML en Java script. in Trello kan je een bord maken met taken er op, als je een taak af hebt kan je de taak weghalen. in Figma kan je een design file en FigJam board maken, een design file is voor het maken van een Wireframe en een FigJam is om een sidemap te maken. met Teams kan je chatten en bellen met klasgenoten, medestudenten en docenten. je kan dingen naar GitHub pushen of dingen van GitHub naar je computer pullen.

er zijn ook meer software die we gebruiken voor andere dingen zoals Eduarte en Canvas
